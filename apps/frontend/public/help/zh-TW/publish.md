# 發佈功能說明

## 概述

發佈功能讓您依照不同使用場景，將問答助手分享給用戶。您可以取得 PWA 網址、iframe 嵌入代碼、Widget 浮動氣泡代碼，並透過網域白名單控制存取權限。

## 發佈助手

### PWA 網址

- **用途**：直接連結到 Chatbot 的完整頁面，適合分享給用戶在瀏覽器中開啟
- **操作**：點擊「複製網址」按鈕，將網址複製到剪貼簿
- **QR Code**：使用手機掃描 QR Code，即可在瀏覽器中打開 Chatbot，並可選擇「加入主畫面」安裝為 PWA 應用

### iframe 頁面嵌入

- **用途**：將 Chatbot 嵌入到您自己的網頁中，作為頁面的一部分
- **操作**：點擊「複製代碼」按鈕，將 iframe 代碼複製到剪貼簿，貼到您的 HTML 頁面中
- **自訂**：可調整 `width` 和 `height` 屬性以符合您的版面需求

### Widget 浮動氣泡

- **用途**：在網站右下角顯示浮動氣泡按鈕，點擊後展開 Chatbot 對話框
- **操作**：點擊「複製代碼」按鈕，將 script 標籤貼到您網站的 `</body>` 前
- **參數**：可透過 `data-*` 屬性自訂位置、大小、氣泡顏色等

## 訪問控制

### 網域白名單功能

啟用後，只有白名單中的網域可以通過 iframe 或 Widget 嵌入此 Chatbot。未在白名單的網域將無法載入 Chatbot。

### 網域格式

- **完整網域**：例如 `example.com`
- **子網域**：例如 `*.subdomain.com` 可匹配 `a.subdomain.com`、`b.subdomain.com`
- **本地測試**：例如 `localhost` 或 `127.0.0.1`

### 操作方式

1. 勾選「啟用網域白名單」
2. 在輸入框輸入網域，點擊「新增」
3. 點擊網域旁的刪除按鈕可移除該網域
4. 變更會自動儲存

## 使用建議

**安全考量**：若 Chatbot 含敏感資訊，建議啟用網域白名單以限制嵌入來源
